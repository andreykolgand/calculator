# Ипотечный калькулятор

Веб-приложение на Flask для расчёта ипотеки: ежемесячный платёж, переплата, график платежей и экспорт в Excel.

## Возможности

- Расчёт аннуитетного графика с учётом первоначального взноса
- Досрочное погашение (уменьшение платежа или срока)
- Визуализация остатка долга (Chart.js)
- Экспорт графика в Excel (.xlsx)
- Адаптивный интерфейс

## Запуск

```bash
python -m venv .venv
.venv\Scripts\activate   # Windows
pip install -r requirements.txt
python app.py
```

Откройте в браузере: http://127.0.0.1:5000

## Стек

- Python 3, Flask
- openpyxl (экспорт Excel)
- Chart.js (графики на фронте)
